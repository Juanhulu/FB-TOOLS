## ⚙️ Cara Install & Jalankan

### 1. Update Terminal
```bash
apt update -y && apt upgrade -y
```
### 2. Install git
```bash
apt install git
```
### 3. Clone repository
```bash
git clone https://github.com/juanhulu/FB-TOOLS.git
```
### 4. Change Directory
```bash
cd FB-TOOLS
```
### 5. Ubah hak akses
```bash
chmod +x main
```
### 6. Run
```bash
./main
```
